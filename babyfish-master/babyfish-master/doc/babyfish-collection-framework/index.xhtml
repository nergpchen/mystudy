BabyFish Collection Framework is a powerful extension of Java Collection Framework, it is the base of ObjectModel

There are three sub modules of BabyFish Collection Framework:

  1. BabyFish X Collection Framework
  2. BabyFish MA Collection Framework
  3. BabyFish Collection Helper
X Collection Framework is the base of MA Collection Framework, MA Collection Framework is an extension of X Collection Framework, so please learn X Collection Framework at first, after that, you can learn MA Collection Framework.

Here are three buttons, each one can let you to goto the document of the corresponding sub module, please select one to learn more.
