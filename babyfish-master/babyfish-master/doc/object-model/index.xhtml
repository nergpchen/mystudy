ObjectModel is the most important functionality of BabyFish, it is the root cause why I created BabyFish.

This is helpful for your data model class design. Often, we design 2 model classes, each one use an association property to retain other one, that is called bidirectional association. If you do nothing anymore, When your program changes the association by one side, but the other side will not be changed correspondingly and automatically, ObjectModel can help you to resolve this problem by annotation declaration, not by code! BabyFish supports two kinds of ObjectModel

ObjectModel4Java      ObjectModel4JPA
